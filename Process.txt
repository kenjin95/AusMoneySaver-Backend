사용자님의 요청대로 **AusMoneySaver** 프로젝트 개발 순서를 군더더기 없이 핵심만 정리해 드립니다.

**1단계: 데이터 수집기 개발 (Backend)**

* **개발 대상:** 환율 크롤링 스크립트 (`scraper.py`)
* **사용 도구:** Python (`requests`, `beautifulsoup4`, `playwright`)
* **목표:** 내 컴퓨터에서 실행 시, 호주 4대 은행 및 핀테크 사이트의 현재 환율을 긁어와 화면에 출력(Print)하는 것까지 성공하기.

**2단계: 데이터베이스 구축 (Database)**

* **개발 대상:** 클라우드 DB 테이블 (`exchange_rates`)
* **사용 도구:** Supabase (무료 티어)
* **목표:** Supabase 프로젝트를 생성하고, 환율 데이터(통화, 금액, 기준시간)를 저장할 테이블 만들기.

**3단계: 수집 및 저장 자동화 (Automation)**

* **개발 대상:** 자동 실행 워크플로우 (`scrape.yml`)
* **사용 도구:** GitHub Actions
* **목표:** 내 컴퓨터가 꺼져 있어도, 30분마다 1단계의 스크립트가 실행되어 2단계의 DB에 데이터가 자동으로 쌓이게 만들기.

**4단계: 웹사이트 화면 개발 (Frontend)**

* **개발 대상:** 환율 비교 웹페이지
* **사용 도구:** Next.js, Tailwind CSS
* **목표:** DB에 쌓인 데이터를 실시간으로 가져와서 "은행 vs 핀테크" 비교표와 계산기를 화면에 띄우기.

**5단계: 배포 (Deployment)**

* **개발 대상:** 실제 접속 가능한 URL 생성
* **사용 도구:** Vercel
* **목표:** 4단계에서 만든 웹사이트를 인터넷에 공개하여 누구나 접속 가능하게 만들기.

**6단계: P2P 기능 추가 (Extension)**

* **개발 대상:** 사용자 간 직거래 게시판
* **사용 도구:** Supabase Auth (로그인), DB (게시글 테이블)
* **목표:** 트래픽이 모인 후, "팝니다/삽니다" 글쓰기 기능 추가.

**가장 먼저 1단계(파이썬 크롤링 코드 작성)부터 시작하시면 됩니다.**
